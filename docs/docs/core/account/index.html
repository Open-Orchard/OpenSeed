Accounts

Current existing functions found in the OpenSeed_Account file. Not all these functions are finalized and there will be changes while we work toward our first release. However, these should represent most of the major features.

#General Account Functions

  • Account Check:

    Checks user accounts returns true if correct.

    example:
    {
    "act":"account_check",
    "devid":"",
    "appid":"",
    "accout":"",
    "passphrase":""
    }

  • Create a User:

    Creates Account based on the included criteria

    example:
    {
    "act"="create_account"
    "devid"="",
    "appid"="",
    "account"="",
    "passphrase"="",
    "email"=""
    }

  • Create a Creator Account:

    Creates Developer account

    example:
    {
    "act"="create_creator_account"
    "devid"="",
    "appid"="",
    "devName"="",
    "contactName"="",
    "contactEmail"="",
    "openseed"=""
    }

  • Create App or Game:

    Creates Application and associates it with the developer.

    example:
    {
    "act"="create_app"
    "devid"="",
    "appid"="",
    "appName"=""
    }

  • Create Profile:

    Creates the long form profile for users,developers, and applications. Called after the ids have been created.

    example:
    {
    "act":"set_profile",
    "devid":"",
    "appid":"",
    "token":"",
    "data1":"",
    "data2":"",
    "data3":"",
    "data4":"",
    "data5":"",
    "type":""
    }

  • Add to History :

    Adds to the history record for the user

    example:
    {
    "act":"update_history",
    "token":"",
    "type":"",
    "appPub":"",
    "data":""
    }

  • Get History :

    Returns the history for a specific account. Optionally you may define a specific application and a total count.

    example:
    {
    "act":"get_history",
    "devid":"",
    "appid":"",
    "account":"",
    "apprange":"",
    "count":""
    }

#Hive Account Functions

  • Send Payment (Hive) :

    Returns the history for a specific account. Optionally you may define a specific application and a total count.

    example:
    {
    "act":"hive_send_payment",
    "devid":"",
    "appid":"",
    "account":"",
    "apprange":"",
    "count":""
    }

  • Flush Keys (Hive) :

    Not needed for most operations as the deletion of keys is a part of every request that requires one. However, when OpenSeed creates an account for a user the keys are stored for fast access. This function can be used to remove the keys from our server.

    example:
    {
    "act":"hive_flush_keys",
    "devid":"",
    "appid":"",
    "hiveaccount":"",
    }

  • Get Account (Hive):

    retrieves the users profile from hive node

    example:
    {
    "act":"get_hive_account",
    "devid":"",
    "appid":"",
    "account":"",
    }

  • Get Full Account (Hive):

    retrieves all user info from hive node

    example:
    {
    "act":"get_hive_account",
    "devid":"",
    "appid":"",
    "account":"",
    }
