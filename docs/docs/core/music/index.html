Music

The Music features found in OpenSeed were designed to support the Steem blockchain (now Hive) and IPFS. In the future this will be the basis for features integerated on multiple chains that use IPFS for storage and our own file management protocols.

info

As a part of the service the original files, which can be any format (aac,ogg,mp3,wav,flac), are re-encoded into a high quality Ogg Vorbis file which is a standard format in many game engines.

#Functions

#General Functions

Functions that return lists or other information used to build prerequisite information for other functions.

Get Genres

Used to get a list of genres found on OpenSeed. This will span all services that OpenSeed connects to as long as the information is available.

Call:

msg='{
"act":"get_genres",
"devPub":"<devPub>",
"appPub":"<appPub>"
}'

Returns:

  • Success

    {"genres":[genre list]}

Get New Artists

Returns a list of artists based on there latest contributions.

note

This is a bit of a misnomer as it will return existing artists with new tracks as well as new artists with new tracks.

Call:

msg='{
"act":"get_new_musicians",
"devPub":"<devPub>",
"appPub":"<appPub>"
}'

Returns:

  • Success

    {"new_musicians":[artits list]}

#Track Functions

These functions return JSON formated string data. For reference the fields and discriptions are as follows.

Field NameDiscription
authorAccount either openseed or the service the music came from.
titleTitle of the track.
postURL associated with the track (if available)
imgIPFS Hash of the image associated with the tack (if available)
curationNot currently used
dateDate added to OpenSeed
oggIPFS hash of the ogg file
typevariable based on source. Currenlty "original" or "cover"
genrelist of genres
tagsvariable tags associated with the track that don't fit "genre" but may include sub genres
durationTime in seconds the track plays.

Get Tracks

Call:

msg= '{
"act":"get_tracks",
"devPub":"<devPub>",
"appPub":"<appPub>",
"start":0 or offset,
"count":number of tracks returned
}'

Returns

  • Success

    {"tracks":{"total":"number returned","results":[song list]}}
